# Changelog — business-model-architect

## v1.0.0 — 2026-07-01
- Initial release. Designs a business model for a brand-new / zero-to-one idea from scratch.
- Workflow: boundary check → tight intake → diverge into distinct archetypes → pre-mortem each → converge on ONE recommended model → de-risk.
- Output: recommended model (who-pays / pricing / GTM / moat) + why-it-wins + runner-up & tradeoff + riskiest assumption + cheapest validation, with unverified inputs labeled.
- Boundary: redirects to `business-model-enhancer` when a revenue model already exists.
- Ambition (bootstrapped vs venture) is a first-class intake dimension that steers the optimization target and the pre-mortem lens.
- Ships 3 golden examples (thin-input intake, audience-but-no-model, existing-model redirect).
